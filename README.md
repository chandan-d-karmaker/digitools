# 🌐 DigiTools
## 🚀 Overview

**Digitools** is a collection of essential digital utility tools designed to simplify common web-based tasks. Built with a focus on speed, responsiveness, and a clean user experience.



## 🧾 Description

DigiTools helps users perform various digital tasks efficiently without switching between multiple apps. The focus of this project is **simplicity, speed, and usability**, making it perfect for both casual users and developers.



## Tech Stack

*  React.js
*  Tailwind CSS
*  JavaScript (ES6+)
*  HTML5 & CSS3


## Key Features

* **Unit Converters:** Length, weight, and temperature conversions.
* **Text Utilities:** Word counting, case switching, and formatting.
* **Calculation Tools:** Math utilities for everyday problem-solving.
* **Modern UI:** A clean, responsive interface that works on all devices.
* **Theme Support:** Built-in Dark/Light mode using DaisyUI themes.

### All-in-One Toolkit

Access multiple useful digital tools in a single platform.

### Clean UI/UX

Minimal, responsive, and user-friendly design for smooth experience.

### Fast Performance

Optimized for speed with lightweight architecture.

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/chandan-d-karmaker/digitools

# Navigate to project folder
cd Folder-name

# Install dependencies
npm install

# Run the project
npm run dev
```
